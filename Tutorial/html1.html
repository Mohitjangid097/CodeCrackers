HTML Tutorial
Welcome to the world of HTML! In this course, we will introduce you to the exciting and essential language of web development: HTML. You will learn how to create and design web pages from scratch, starting with the basic structure, and eventually progressing to more advanced features and styling. By the end of this course, you will have a solid understanding of HTML and the confidence to create your own web pages. HTML, or Hypertext Markup Language, is the backbone of web content. It provides the structure for websites and is used in conjunction with other technologies, such as CSS (Cascading Style Sheets) and JavaScript, to create fully functional and visually appealing websites. HTML is the foundation of web development, and mastering it is an essential step in becoming a web developer or designer.
What is HTML
HTML (Hypertext Markup Language) is the standard markup language used to create and design web pages. It consists of a series of elements that define the structure and content of a webpage. HTML uses tags, which are enclosed in angle brackets, to create elements like headings, paragraphs, lists, and images. HTML is based on a tree-like structure, starting with the root element (the tag) and branching out into various nested elements (like head, body, h1, p, etc.). These elements and their associated tags define the content, structure, and semantics of a web page.

Example of a simple HTML page:


< !DOCTYPE html>
< html>

< head>
< title> My First HTML Page < /title>
< /head>

< body>
< h1> Welcome to my website! < /h1>
< p> This is a paragraph. < /p>
HTML Editors
HTML editors are tools that help you write and edit HTML code. There are two main types of HTML editors: text editors and WYSIWYG (What You See Is What You Get) editors. Text editors, such as Notepad++, Sublime Text, and Visual Studio Code, provide a simple and efficient way to write HTML code. They often include syntax highlighting, code completion, and other features that make writing HTML easier and more efficient. These editors are popular among developers because they offer more control over the code and tend to be lighter and faster than WYSIWYG editors.
WYSIWYG editors, such as Adobe Dreamweaver, allow you to design web pages visually, without having to write the HTML code manually. These editors generate the HTML code for you based on the visual layout you create. While WYSIWYG editors can be more user-friendly for beginners, they may produce messy or inefficient code, and they can limit your understanding of the underlying HTML structure.
Basic HTML Document Structure
An HTML document consists of several key components, each with its own role in defining the structure and content of the web page. The basic structure of an HTML document includes the following elements:
DOCTYPE html This declaration defines the document type and version of HTML being used. It should always be the first line of your HTML document.
html The root element of the page, which contains all other elements.
head This element contains meta-information about the document, such as the title, character encoding, and links to external resources like CSS and JavaScript files.
title Defines the title of the web page, which is displayed in the browser's title bar or tab.
body Contains the content of the web page, such as text, images, and links.
meta charset="UTF-8" This element sets the character encoding for the document, ensuring that all characters display correctly. In this case, we're using UTF-8, which is the most common encoding for web pages.
link rel="stylesheet" href="styles.css" This element links an external CSS file (in this case, "styles.css") to the HTML document, allowing you to style the content of the web page.
script src=" ": This element links an external JavaScript file (in this case, "script.js") to the HTML document, enabling you to add interactivity and dynamic content to the web page.
h1: Represents a top-level heading on the web page. HTML headings range from h1 (the most important) to h6 (the least important).
p: Represents a paragraph on the web page. This element is used to define blocks of text separated by a line break.

Here's an example of a basic HTML document structure:


< !DOCTYPE html>
< html>

< head>
< meta charset="UTF-8">
< title> My First HTML Page < /title>
< link rel="stylesheet" href="styles.css">
< /head>

< body>
< h1> Welcome to my website! < /h1>
< p> This is a paragraph. < /p>
< /body>

< /html>
HTML Elements
HTML elements are the building blocks of web pages, and they provide structure, content, and meaning to the page. Each element is defined by a pair of tags, an opening tag and a closing tag, which enclose the content of the element. The tags are written within angle brackets (<>), and the closing tag includes a forward slash (/) before the element name. Here are some common HTML elements and their descriptions: h1 to h6 These elements represent headings, with h1 being the most important and h6 being the least important. Headings are used to organize content on the page and improve readability.

Example:


< h1 > Main Heading < /h1 >
< h2 > Subheading < /h2 >
< h3 > Sub-subheading < /h3 >


p: Represents a paragraph. This element is used to define blocks of text separated by a line break
ul and ol: Define unordered (bulleted) and ordered (numbered) lists, respectively. These elements are used in conjunction with the li (list item) element.
a: Represents a hyperlink, which allows users to navigate to other web pages or resources.
href: href attribute specifies the destination URL.

These are just a few examples of the many HTML elements available for use in web development. As you progress through the course, you'll learn about additional elements and how to use them effectively to create a well-structured, visually appealing, and accessible web page.
